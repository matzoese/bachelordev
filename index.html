Bachelor Thesis: Example Traced Tutor by Matthias Stas

This is a website where you as a student will be asked to solve multiple computer science related tasks. Those tasks will be chosen randomly, so each student will have a unique experience. There will be 3 different topics and for each topic 3 different difficulty classes, resulting in 9 task pools which contain 3 tasks each. You need to solve 9 tasks, one from each task pool.
